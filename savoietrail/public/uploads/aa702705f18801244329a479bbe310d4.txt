Brief Projet Chef d'oeuvre
==========================

Le projet chef d'oeuvre est un projet individuel qui se déroulera sur 5 semaines du 29/07 au 30/08.

Rappel : le projet chef d'oeuvre est un outil qui permet de valoriser les compétences acquises lors du passage du titre professionnel.

# Objectifs
Doit reprendre les principales compétences :
1 - Maquetter
2 - Web statique
3 - Web dynamique
5 - Créer une BDD
6 - Composants d'accès à la BDD
7 - Partie Back-end

# Description
### Partie publique:
Application web comportant une cartographie de point de vue ou lieu d'interet pour les randonneurs et des trajet de ballade et randonner de différente difficulter, téléchargement des parcours sur son telephone (au cas ou plus de réseau). l'application comportera des photos des lieux

### Partie utilisateur: 
Le randonneurs pourra s'inscrire pour pouvoir ajouter des trajets, mettre des trajets en favori et laisser des commentaire.

### Partie back-end:
Gestion d'utilisateur, gestion des données utilisateur, calcul d'itinéraire, traitement du contenu liée aux parcours(comment liée).

# Bonus
* 1. Espace de commentaire ou avis pour les parcours.
* 2. Systeme de favori pour les utilisateurs
* 3. Systeme de notation.
* 4. Widget météo
* 5. Bot automatiser et mise a jour automatique


# Description du projet
Pour la suite, nous diviserons le projet en 5 modules:

* 1. Base de données (bot)
* 2. Gestion de compte
* 3. Contenu
* 4. Gestion du upload des parcours

## Module 1 : la gestion de compte utilisateur 
### Partie 1:
Créer la BDD 
### Partie 2:
Il s'agit de proposer une interface de gestion de compte par utilisateur permettant d'implémenter de nouveau parcour.
Ce module permetra a l'utilisateur d'implémenter des parcours, ces parcours seront liée a sont compte visible par tout le monde et pourra etre commentée ou noté(bonus).
### partie 3:
Créer la partie admin et publique 

## Module 2 : les données de parcour (format des parcours gpx)
Les données initial sont issues d'un bot qui récupera un maximum de donner sur 2 a 3 site web et implémentera la BBD. Il s'agi de traiter correctement les données pour pouvoir impléménter la BDD de parcour et poin de vue ou lieu d'intéret.
 

## Module 3 : Contenu
Ajout du contenu et le Lier aux donnnées de parcour.

## Module 4 : upload de parcours
Téléchargement du parcour avec sa description au format pdf.
